养锅秘籍：把不锈钢锅变不粘锅

发表于 2016-09-09

为了健康，我们家都改用不锈钢锅。可是你有没有不锈钢锅会粘锅的烦恼？像煎蛋，煎鱼，真的是一下锅马上粘，所以后来只好不锈钢和不粘锅混用，煎蛋煎鱼时用不粘锅，炒菜用不锈钢锅。可是不粘锅使用一段时间后，表面因为受损，开始粘锅，这时候也表示表面的化学物质已经剥落了。于是又不免担心不知道吃进去多少致癌物质了！
这样的烦恼你一定也有吧？
如果有，今天你就要好好看这篇文章。
将不锈钢锅变身不粘锅，其实不难，关键在养锅秘笈，而且养过一次锅以后，如果正确使用，可以撑一段很久的时间。
养锅步骤

  1. 这是一个不锈钢锅，清洗干净。

  2. 开火加热锅子。同时倒入一点白醋 （只要是食用白醋都可以），持续加热到接近滚烫，关火。

  3. 然后准备一个钢刷，用滚烫的醋把锅子内面整个用力刷洗（多垫一块海绵布这样刷才不会烫到手）。
    这个步骤要仔细刷，每一个地方 （尤其是煎东西的接触面） 都要用力刷到。这样可以除去不锈钢毛细孔内的污垢和杂质。刷干净后倒掉白醋，用清水冲干净。

  4. 然后用大火加热锅子，加热到所有的水滴都蒸发掉，再继续加热一下下，让锅子很热 （不要热到把锅子烧坏了）。

用一点点水倒进去试验锅子有多热，那水珠直接已经在锅面滚动，有点像水银一样（上图右），这样锅子就很够热了。

  1. 这时候转中小火，同时倒入色拉油 （或其他你平常在炒菜的食用油），然后不断将锅子摇一摇、转一转。确保锅内面都有粘到油，此时锅子都用中小火保持温度，至少让油在锅子里面待两三分钟。这个过程就是要不锈钢锅表面的毛细孔可以吸进这些油，达到不粘锅的效果。

差不多以后就关火，这时候养锅就算完成。
你如果需要煮菜，就可以开始煮了。
如果不需要立刻就用来煮菜，等冷却后把油倒掉、用水和洗洁精把锅子洗干净。把锅子擦干的时候，你会发现此时锅子感觉非常油亮。
每次使用前需受热均匀
请注意，每次煎东西时，仍然先开中小火，冷油下去以后，保持中小火加热油约两分钟。
顺便把锅子动一动，让油均匀的粘附在锅子内面，这个步骤以后，就可以将火开到你需要的强度，像平常一样煎东西炒菜了。
注意，提前让锅内的油均匀受热很重要。在美国线上厨艺学校ROUXBE的“What happen when heating a pan”影片中，指出当不锈钢锅加热时，金属分子受热后会开始向各个方向膨胀，在锅子表面的“孔”(pore)会受到推挤，造成部分”孔”闭合部分”孔”打开，如果在这个时候放入食物尤其是油脂含量较少的鱼肉与鸡胸肉，便会发生粘锅现象！

当锅子加热到适当的温度时，“孔”被推挤的现象消退(subside)，粘锅现象便不会发生！以此原理来看“养锅”，推测养锅的过程是让锅子表面的“孔”均匀张开，当食物锅子表面的个孔对食物的附着大小均匀，就不会产生粘锅的现象。
不锈钢变身不粘锅，实际效果展示！
来示范一下煎荷包蛋，看看是不是真的那么神奇。
将一颗蛋打下去，等蛋白略成形了；

紧张紧张，用铁铲稍从蛋白边边掀一下，恩，已经成形了，于是摇动锅子…….
嘿~荷包蛋果然在锅子内面溜来溜去，你看看你看看~！

第一颗蛋油用得稍多，所以把油倒干净一点，再煎第二颗蛋。
等蛋白成形后，用锅铲稍微在边边又掀一下，哈，果然还是不粘。

轻轻松松煎出两颗完整的荷包蛋，你说是不是太神奇了？

最后把锅子像平常一样清洗完，锅内面仍旧保持亮亮的油光，不粘的效果仍在。
当然要使用一段时间后才知道可以保持多久的不粘效果，不过通常都可以撑好一阵子。
难怪以前看到快炒店的锅子，只是铁锅，可都很“油亮”，而且都不粘。好像他们都知道养锅的秘诀。大家要学起来喔！
为了你和你家人的健康，这样就可以不需要再使用市面上的特氟龙不粘锅，不用担心不知道吃进去多少致癌物质。而且不锈钢锅通常都很耐用，可以省下经常更换不沾锅的钱哦！

如何用VPS搭建PPTP VPN

发表于 2016-03-17

内容目录

目录

[TOC]

环境
: ubuntu14.04 32位

1、下载安装

apt-get install pptpd

2、配置/etc/pptpd.conf

vim /etc/pptpd.conf

添加下面两行（在配置文件的最后，取消注释修改IP即可）

localip 172.16.17.63 #这个就是当前主机的IP地址
remoteip 10.0.0.2-100 #这个是给客户端分配IP的地址池，可通过这个配置限制连接数

3、添加DNS

cd /etc/ppp
vim options

ms-dns设置为8.8.8.8

4、添加服务器名称

vim pptpd-options
name 'server-name' #服务器名称

5、配置客户端登录的用户名和密码

vim chap-secrets
用户名 服务器名 密 码 允许登录的IP
vpn * asdf *

服务器名和允许登录的IP可以用通配符

6、查看运行端口

netstat -tnulp | grep pptpd

可以看到1723端口开放
修改内核设置，使其支持转发

echo 1 > /proc/sys/net/ipv4/ip_forward

要想永久生效，设置

vim /etc/sysctl.conf
net.ipv4.ip_forward=1

将后面的值改为1，然后保存文件
使设置马上生效

sysctl -p

7、添加访问外网的配置

iptables -t nat -A POSTROUTING -s 10.0.0.0/24 -o '主机连接外网的网卡名' -j MASQUERADE

8、重启pptpd服务

/etc/init.d/pptpd restart
